# Assets 目录

此目录存放项目相关的图片资源。

## 目录结构

```
assets/
├── gallery/                    # 展示画廊
│   ├── example-01-grain-oil.png   # 示例 1：中国粮油价格波动
│   ├── example-02-placeholder.png # 示例 2：待添加
│   ├── example-03-placeholder.png # 示例 3：待添加
│   └── example-04-placeholder.png # 示例 4：待添加
├── wechat-group-qr.png        # 微信群二维码（请替换为真实二维码图片）
└── logo-placeholder.png        # 项目 Logo（请替换为真实 Logo）
```

## 如何添加展示图片

1. 在 Midjourney 中使用 `examples/` 目录下的提示词生成图表
2. 下载生成的图片
3. 放入 `assets/gallery/` 目录
4. 更新 `README.md` 中对应展示区的图片路径
5. 替换 `wechat-group-qr.png` 为真实的微信群二维码图片
